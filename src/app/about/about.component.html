Angular Express is a starter project that implements best practices in coding, building and testing Angular apps with NodeJS Express.

Features

  • Ready to go, statically typed build system using Angular CLI for working with TypeScript.
  • Production and development builds.
  • Sample unit tests with Jasmine and Karma.
  • End-to-end tests with Cypress.
  • Development server with live reload.
  • TypeScript definition management using Types.
